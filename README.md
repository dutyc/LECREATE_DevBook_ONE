## 📜 项目宣言与哲学

> **在 AI 时代，不是"AI 赋能生活"，而是“生活赋能 AI"。**  
> 让每个人享受科技的定制化、便利化，让人们可以全身心地投入对生命本源的探索。

**LECREATE DevBook ONE** 不是一台普通的笔记本，而是一场关于**创造者自由**的实践。它诞生于对 x86 笔记本商品化困境的反思，成长于 ARM 开源生态的繁荣土壤，最终将回归到每一位开发者手中——成为你们定制、修改、创造的起点。

### 为什么开源？为什么不量产？

> **对原型机所有建造资料的开源，是作为后辈晚生献给开源社区的一份回报。**
>
> 就像 Linux 开源内核而吸引成千上万开发者共同维护一样，DevBook ONE 的硬件原型设计需要更多的开发者的贡献才能得到繁荣，才能让每位开发者都有定制自己的"超级本"的能力。
>
> 有人问我为什么不创立自己的公司去量产、销售"超级本"？
>
> 1.  **一方面是出于对开源社区的回馈**；
> 2.  **另一方面，量产原型机就失去了项目初心**——让每位开发者拥有定制修改的能力。
>
> 把原型机复制上百台量产，即便有人想购买，有人愿意下单，但他们缺失了最关键的一个环节——**将自己的创意变为现实的过程**。他们买到量产本，买到的只是"我的创意"，这就违背了项目初心，违背了我的最初目标。
>
> **我的目标不是卖给你一台电脑，而是赋予你创造自己电脑的能力。**

### 一人成军，回归本源

我不愿意被"前端工程师"、"内核维护者"、"PCB 设计者"等标签所约束。**我一个人就是一个团队，一人成军。** 这不是因为我不善与人交流，而是出于对生活的热爱，对这一生灵魂终极目标的追求，对回归本源的向往。

做这个项目，是为了向世人证明：**让生活赋能于 AI，让每个人享受科技的定制化。** 我不希望大家将我称为"天才开发者"或"大佬"，我只是在我人生中的一个阶段做了一件小小的作品。只是想让大家明白"生活赋能于 AI"的正确观念，明白人生终极目标、生命真相、魂归故里、天人合一。

## 🎯 项目概述

| 项目信息     | 详情                                   |
| ------------ | -------------------------------------- |
| **项目名称** | LECREATE DevBook ONE                   |
| **开发状态** | 🔴 **独立开发中**（原型机未完成）       |

**LECREATE DevBook ONE** 是一台基于 **Rockchip RK3588** 架构的终极全栈开发与运维终端。它集成了原生硬件实验室、容器化 KVM 控制台、蜂窝网络连接与极致能源系统于一体。

*   **核心理念**：软件定义硬件 · 原生实验室 · 移动数据中心 · 全域连接
*   **目标用户**：嵌入式工程师、Linux 内核开发者、系统运维专家、物联网创客、极客玩家

## ✨ 核心特性

### 🔧 1. 原生硬件实验室 (Native Hardware Lab)

*让笔记本本身成为开发工具，告别转接线和面包板。*

| 功能模块             | 详细描述                                                     | 应用场景                                                |
| -------------------- | ------------------------------------------------------------ | ------------------------------------------------------- |
| **板载 USB-TTL**     | 集成高性能串口芯片，直接引出 `TX/RX/GND` 排针。**软件切换 3.3V/5V 电平**，软件开关 VCC 电源。 | 调试路由器、单片机、交换机 Console 口，无需外接转接器。 |
| **多功能 GPIO 排针** | 直接引出 SoC 的 `GPIO`, `I2C`, `SPI`, `PWM`, `ADC`, `UART`。标准 2.54mm 间距，杜邦线直插。 | 直接连接传感器、OLED 屏、逻辑分析仪，快速原型验证。     |
| **可编程状态灯**     | 位于掌托或边框的 RGB/单色 LED，由 SoC 的 PWM GPIO 控制。支持编译完成提示、负载可视化。 | 系统状态直观反馈，极客浪漫装饰。                        |
| **扩展接口区**       | 预留 I2C/SPI 专用扩展座，支持直接插拔小型模块。              | 教学演示、快速测试、模块化扩展。                        |

### 🖥️ 2. 容器化 KVM 控制台 (Containerized KVM Console)

*内置带外管理能力，一台机器拯救所有变砖设备。*

| 功能模块            | 详细描述                                                     | 技术实现                                           |
| ------------------- | ------------------------------------------------------------ | -------------------------------------------------- |
| **USB Gadget 模拟** | 利用 RK3588 USB-C 控制器，模拟成 **USB 键盘/鼠标 (HID)**、**USB 网卡**、**USB 存储**。 | Linux Kernel Configfs USB Gadget + 主线驱动。      |
| **HDMI 采集集成**   | 机身集成或侧边扩展 **USB 3.0 HDMI 采集卡**，支持 1080P@60fps 低延迟输入。 | UVC 驱动 + FFmpeg/GStreamer 硬解码。               |
| **oneKVM 容器**     | 基于开源 **oneKVM** 项目，Docker 容器化部署，隔离底层硬件与用户界面。 | `docker run --privileged --device /dev/video0 ...` |
| **浏览器套壳应用**  | 定制 **Electron/Tauri** 应用，全屏加载 oneKVM Web 界面，隐藏浏览器 UI，提供原生应用体验。 | 全局快捷键、一键启动、配置文件管理。               |

### 🔋 3. 能源与电源系统 (Power & Energy)

*重新定义移动能源，兼具超级充电宝功能。*

| 功能模块         | 详细描述                                                     | 核心优势                               |
| ---------------- | ------------------------------------------------------------ | -------------------------------------- |
| **异构电池组**   | 采用多节**手机软包锂电池**堆叠，填满机身空隙。目标容量 **80Wh - 99Wh**。 | 极致空间利用率，续航 **15-20 小时+**。 |
| **双向 PD 3.1**  | 全功能 USB-C 口：**输入** 100W 快充；**输出** 充电宝模式 100W；**旁路供电** 保护电池。 | 一机多用，户外应急电源。               |
| **智能 BMS**     | 高精度电量计，实时上报电量/电流/温度/健康度至系统。支持多串平衡充电。 | 数据透明，安全可靠，软件可监控。       |
| **深度电源管理** | 基于主线内核的 Power Domain 控制，闲置外设彻底断电。         | 极致待机（>10 天），低功耗运行。       |

### 💾 4. 存储与计算架构 (Compute & Storage)

*高性能 ARM 架构，支持模块化存储隔离。*

| 功能模块         | 详细描述                                                     | 核心价值                                              |
| ---------------- | ------------------------------------------------------------ | ----------------------------------------------------- |
| **SoC 核心**     | **Rockchip RK3588**：8 核 CPU, 6TOPS NPU, 8K 编解码，2×PCIe 3.0。 | 性能强悍，能效比极高，AI 算力充足。                   |
| **大内存**       | **16GB / 32GB LPDDR4X/LPDDR5**。                             | 满足大型编译、Docker、虚拟机、多任务需求。            |
| **热插拔硬盘笼** | 侧边专用仓或高速 USB-C 接口，基于 **NVMe 转 USB 3.2 Gen2** 方案。 | **环境物理隔离**：不同硬盘装不同系统/项目，即插即用。 |
| **TF 卡槽**      | SDIO 3.0 接口，支持热插拔。                                  | 快速启动备用系统，数据交换，应急启动。                |

### 📡 5. 原生蜂窝网络连接 (Native Cellular Connectivity)

*随时随地，想连就连，真正的全场景覆盖。*

| 功能模块           | 详细描述                                                     | 技术要点                         |
| ------------------ | ------------------------------------------------------------ | -------------------------------- |
| **M.2 Key-B 插槽** | 板载标准 M.2 Key-B 接口，兼容 4G LTE / 5G Sub-6GHz 模块。    | PCIe/USB 双模式，未来可升级 5G。 |
| **天线设计**       | 预留 **I-PEX 天线座**，支持外接高增益柔性天线（FPC）。       | 确保野外信号稳定，射频性能可靠。 |
| **智能切换**       | 支持 WiFi/4G 自动故障转移；可开启 NAT 转发，为外接设备提供互联网接入。 | 运维场景刚需，单兵网关能力。     |

## 🔄 典型工作流 (Workflow Examples)

### 🔧 场景 1：现场救砖

1.  连接变砖开发板 → 板载 TTL 直连
2.  打开「DevBook KVM」应用 → 看到目标设备 BIOS 画面
3.  模拟键盘输入修复引导 → 挂载 ISO 重装系统
4.  **全程无需外接显示器、键盘、U 盘、转接器**

### 🧪 场景 2：嵌入式原型开发

1.  杜邦线直连笔记本 GPIO 排针 → 读取传感器数据
2.  修改设备树调整 PWM 频率 → 编译内核
3.  状态灯呼吸闪烁提示编译完成 → 自动部署到目标设备
4.  **全程无需面包板、逻辑分析仪、外接电源**

### 🌐 场景 3：野外无网环境运维

1.  笔记本通过 4G 蜂窝网络联网
2.  通过 USB/串口连接无网目标设备
3.  开启 NAT 转发，目标设备间接上网
4.  启动 KVM 控制台，远程调试 + 提交代码
5.  **全程无需手机热点、无需寻找 WiFi**

### 🔋 场景 4：超长续航移动办公

1.  早上满电出门 → 全天编码/调试/会议
2.  同事手机没电 → 开启「充电宝模式」快充救援
3.  晚上回家 → 笔记本仍剩 30% 电量
4.  **全程无需携带充电器，能源自由**
